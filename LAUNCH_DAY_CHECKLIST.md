## COMMON ISSUES & SOLUTIONS

### Issue: Patient not found in system
**Solution:**
1. Search by phone number
2. Check spelling of name
3. Create new patient record if truly new
4. Link to any previous appointments

### Issue: Payment terminal down
**Solution:**
1. Use backup terminal
2. Process as "pending" in system
3. Call payment processor
4. Accept cash if necessary
5. Document in daily log

### Issue: Scheduler not loading
**Solution:**
1. Refresh browser (Ctrl+F5)
2. Clear browser cache
3. Try different browser
4. Check internet connection
5. Contact technical support

### Issue: Clinician running behind
**Solution:**
1. Check current appointment status
2. Notify waiting patients (SMS or in-person)
3. Offer rescheduling option
4. Adjust subsequent appointments
5. Document delay reason
