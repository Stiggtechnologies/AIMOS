## Getting Started

### First-Time Login

**Step 1: Access the System**
1. Open your web browser (Chrome, Firefox, Safari, or Edge)
2. Navigate to: https://your-aim-system.vercel.app
3. You'll see the AIM Rehab login screen

**Step 2: Enter Your Credentials**
1. Enter your email address
2. Enter your temporary password (provided by IT)
3. Click "Sign In"

**Step 3: Change Your Password** (First Login Only)
1. You'll be prompted to change your password
2. Enter a strong password (minimum 8 characters)
3. Include uppercase, lowercase, numbers, and special characters
4. Click "Update Password"

**Step 4: Explore the Dashboard**
- Take a tour of the main dashboard
- Familiarize yourself with the navigation menu
- Check your profile settings

### Password Reset

If you forget your password:
1. Click "Forgot Password?" on the login screen
2. Enter your email address
3. Check your email for reset instructions
4. Click the link in the email
5. Enter and confirm your new password

## Navigation & Interface

### Main Navigation Menu

The left sidebar contains your main navigation:

**Core Modules:**
- **Dashboard** - Your personalized home screen
- **AI Assistant** - Intelligent search and assistance

## Core Features by Role

### For All Staff

#### Accessing Announcements
1. Click **Announcements** in the navigation
2. View latest company news and updates
3. Click an announcement to read full details
4. Filter by category: General, HR, Clinical, IT
5. Search announcements using the search bar

#### Using the Document Library
1. Click **Documents** in the navigation
2. Browse documents by category
3. Use the search bar to find specific documents
4. Click a document to view or download
5. Filter by type: Policies, Procedures, Forms, Templates

#### Viewing Clinic Information
1. Click **Clinics** in the navigation
2. See all clinics you have access to
3. Click a clinic name for details
4. View: Address, Hours, Services, Staff, Performance

#### Using the People Directory
1. Click **People** in the navigation
2. Browse or search for colleagues
3. Click a name to view profile
4. See: Role, Contact info, Clinic assignment
5. Use filters: Clinic, Role, Department

#### Accessing SOPs
1. Click **SOPs** (Standard Operating Procedures)
2. Browse by category or search
3. Click to view SOP details
4. Download PDF version if needed
5. Check version number and approval status

### For Clinicians

#### Using Clinician Mobile Dashboard

**Accessing Your Schedule**
1. Click **Clinician Mobile** in navigation
2. View today's appointments automatically
3. See patient name, time, appointment type
4. Check appointment status

**Checking In a Patient**
1. Find patient's appointment in today's list
2. Click **Check In** button
3. Confirm patient arrival
4. Status updates to "Checked In"
5. Timestamp recorded automatically

**Checking Out a Patient**
1. Find patient's checked-in appointment
2. Click **Check Out** button
3. Status updates to "Completed"
4. Checkout time recorded

**Creating Quick Notes**
1. Click **Quick Notes** tab
2. Click **Create Note** or **Add Note** from appointment
3. Select note type: Clinical, Administrative, Follow-up
4. Enter note text
5. Add tags if needed (optional)
6. Click **Save Note**
7. Note appears in your history

**Managing Your Availability**
1. Click **Availability** tab
2. View your current weekly schedule
3. Click **Add Availability** to set new hours
4. Select day of week
5. Set start and end times
6. Add notes if needed
7. Click **Save**

**Best Practices for Clinicians:**
- Check in patients promptly upon arrival
- Create notes during or immediately after appointments
- Review tomorrow's schedule at end of each day
- Update availability changes at least 48 hours in advance
- Use tags for easy note retrieval

## Module-Specific Training

### AI Assistant

**How to Use the AI Assistant**
1. Click **AI Assistant** in navigation
2. Type your question in natural language
3. Examples:
   - "Show me revenue for Calgary clinic this month"
   - "Who are the top referrers this quarter?"
   - "What's our patient satisfaction score?"
   - "List all clinicians at Edmonton clinic"
4. Review the AI-generated response
5. Ask follow-up questions for more details
6. Export results if needed

**Best Practices:**
- Be specific in your questions
- Use proper clinic and service names
- Specify time periods when relevant
- Ask follow-up questions to drill deeper
- Verify critical data from source systems

**What the AI Assistant Can Do:**
- Answer questions about clinic data
- Provide financial summaries
- Generate reports on demand
- Search across all modules
- Explain metrics and trends
- Suggest next actions

### Operations Engine

**Case Aging Workflow**
1. Navigate to **Operations** > **Case Aging**
2. View cases organized by age
3. Color coding:
   - Green: Within SLA
   - Yellow: Approaching threshold
   - Red: Overdue, needs attention
4. Click a case to view details
5. Take action: Escalate, Resolve, or Add Note
6. System sends automatic alerts based on rules

**Capacity Planning**
1. Navigate to **Operations** > **Capacity**
2. View current capacity by clinic
3. See available appointment slots
4. Monitor utilization rates
5. Identify constraints
6. Plan staffing adjustments

**Credentials Tracking**
1. Navigate to **Operations** > **Credentials**
2. View all staff credentials
3. See expiration dates
4. Filter by:
   - Expiring soon (within 30/60/90 days)
   - Expired
   - Clinic
   - Credential type
5. Click to view details or upload renewals
6. System sends automatic renewal reminders

### Clinic Launch Management

**Creating a New Clinic Launch**
1. Click **Clinic Launches**
2. Click **New Launch Project**
3. Enter clinic details:
   - Clinic name
   - Location
   - Target open date
   - Launch type (De Novo or Partner)
4. Select project template
5. Assign project manager
6. Click **Create Project**

**Managing Launch Phases**
1. Open launch project
2. View current phase and status
3. Phase gates:
   - Phase 1: Planning & Site Selection
   - Phase 2: Build-Out & Staffing
   - Phase 3: Training & Licensing
   - Phase 4: Soft Launch
   - Phase 5: Grand Opening
4. Complete tasks within each phase
5. Submit for gate approval
6. Progress to next phase upon approval

**Tracking Milestones**
1. View milestone timeline
2. Mark milestones complete as achieved
3. Update dependencies
4. Flag blockers or issues
5. Generate status reports for stakeholders

### Partner Clinics

**Viewing Partner Clinic Dashboard**
1. Click **Partner Clinics**
2. View all partner clinic relationships
3. See partnership status and health score
4. Monitor key metrics:
   - Patient volume
   - Revenue share
   - Quality scores

**Managing Partner Clinic Launches**
1. Access partner clinic launch project
2. Follow 90-day launch plan
3. Track weekly milestones
4. Coordinate with partner contact
5. Monitor integration progress
6. Review partnership agreements

## Best Practices

### General System Use

**Do's:**
- Log out when leaving your workstation
- Keep your password secure and private
- Report any suspicious activity immediately
- Use appropriate channels for different communication types
- Keep your profile information up to date
- Document important actions and decisions
- Use search features before asking colleagues

**Don'ts:**
- Share your login credentials
- Leave your computer unlocked
- Access patient data you don't need for your work
- Take screenshots of sensitive information
- Use personal email for work communications
- Bypass security protocols
- Ignore system alerts or notifications

### Data Entry

**Accuracy:**
- Double-check all data before saving
- Use standard formats (dates, phone numbers, etc.)
- Complete required fields
- Use dropdown options when available (ensures consistency)
- Review auto-filled data for accuracy

**Timeliness:**
- Enter data as close to real-time as possible
- Don't delay documentation
- Update status changes promptly
- Complete daily tasks before end of day

**Completeness:**
- Fill in all relevant fields
- Add notes when context is needed
- Attach supporting documents
- Tag entries appropriately

### Privacy & Security

**HIPAA Compliance:**
- Access only patient data necessary for your role
- Log out after viewing patient information
- Don't discuss patient information in public areas
- Secure printed patient information
- Report any privacy breaches immediately

**Data Security:**
- Use strong, unique passwords
- Don't write down passwords
- Be cautious of phishing emails
- Verify requests for sensitive information
- Report lost or stolen devices immediately

## Troubleshooting

### Common Issues

#### "I can't log in"
**Solutions:**
1. Verify you're using the correct email address
2. Check Caps Lock is off
3. Try password reset
4. Clear browser cache and cookies
5. Try a different browser
6. Contact IT support if issue persists

#### "I don't see a clinic/patient/document I'm looking for"
**Solutions:**
1. Check your permissions with your manager
2. Verify spelling in search
3. Check filters are not too restrictive
4. Try global search (⌘K or Ctrl+K)
5. Ensure you have access to that clinic
6. Contact IT if you should have access

#### "The page is loading slowly"
**Solutions:**
1. Check your internet connection
2. Close unnecessary browser tabs
3. Clear browser cache
4. Try a different browser
5. Refresh the page
6. Report if problem persists

#### "I made a mistake in data entry"
**Solutions:**
1. Navigate to the record
2. Click Edit (if you have permission)
3. Correct the error
4. Save changes
5. If you can't edit, contact your manager or IT
6. For critical errors, report immediately

#### "I'm not receiving notifications"
**Solutions:**
1. Check notification settings in your profile
2. Verify email address is correct
3. Check spam/junk folder
4. Ensure browser notifications are enabled
5. Contact IT if notifications are still not working
