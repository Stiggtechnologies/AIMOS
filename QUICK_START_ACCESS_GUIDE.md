## 🔧 HOW TO USE CORE FEATURES

### SCHEDULING AN APPOINTMENT

1. Navigate to **Scheduler**
2. Click on desired time slot
3. Fill in patient details:
   - Search existing patient or create new
   - Select service type (Initial Assessment, Follow-up, etc.)
   - Choose treatment room
   - Assign clinician
4. Click **Book Appointment**
5. System automatically sends confirmation

### PATIENT INTAKE (NEW PATIENT)

1. Navigate to **CRM > Intake Pipeline**
2. Click **New Lead**
3. Enter patient information:
   - Name, phone, email
   - Referral source
   - Chief complaint
   - Insurance type
4. Click **Convert to Patient** when ready to book
5. System creates patient record and allows booking

### CLINICAL DOCUMENTATION

1. Navigate to **Scheduler**
2. Click on patient appointment
3. Click **Start Charting**
4. Complete SOAP note:
   - Subjective findings
   - Objective measurements
   - Assessment and diagnosis
   - Plan and treatment
5. Use **CDS Recommendations** for evidence-based guidance
6. Click **Save & Sign**

### ACCESSING EVIDENCE-BASED PROTOCOLS

1. Navigate to **AIM OS > Clinical Intelligence**
2. Use **Search** to find condition (e.g., "concussion")
3. View **Starter Packs** with:
   - Assessment protocols
   - Treatment guidelines
   - Exercise progressions
   - Evidence citations
4. Click **Apply to Patient** to use in documentation

### FINANCIAL REPORTING

1. Navigate to **AIM OS > Financial View**
2. Select date range
3. View metrics:
   - Revenue by clinic
   - Collections vs. charges
   - Accounts receivable aging
   - Expense tracking
4. Click **Export Report** for detailed analysis

### LAUNCH NEW CLINIC

1. Navigate to **Launches**
2. Click **New Launch Program**
3. Select template (e.g., "Partner Clinic Launch")
4. Fill in clinic details
5. System generates:
   - 90-day project plan
   - Phase gate checklists
   - Task assignments
   - Resource requirements
6. Track progress through phase gates

## 🆘 TROUBLESHOOTING

### Can't Log In

1. Verify email address is correct
2. Use password reset link
3. Check with system administrator
4. Verify user account is active

### Can't See Specific Clinic Data

1. Check your clinic access permissions
2. Navigate to correct clinic using domain switcher
3. Verify role has appropriate permissions
4. Contact administrator for access

### Missing Features

1. Check your role permissions
2. Some features are role-specific
3. Navigate to correct module
4. Verify feature is enabled for your clinic

### Performance Issues

1. Clear browser cache
2. Try different browser
3. Check internet connection
4. Report persistent issues
